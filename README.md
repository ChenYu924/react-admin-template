# 使用到的技术栈：
主框架：React、react-router-dom、基于rtk的redux和Ant Design UI框架。
<br>
样式表：scss、tailwindcss
<br>
类库：ahooks、animate.css、decimaljs、echarts、js-cookie...
<br>
支持：TypeScript
